# sample-app

## サービス

AI目的特化支出管理アプリ

「サブスクや外食など“特定のムダ出費”だけをAIが見張って、削減案まで教えてくれる支出管理アプリ」

## 機能

- 家計簿の登録
- 可視化
- AIの推奨（固定費の削減）

## 利便性

- 登録が楽
- 可視化が見やすい
- AIを使って費用を減らせる

## 便利機能

- 細かいカスタマイズ性

### 競合差別化

- 連携しないことによる安心さ
- 家計簿全体ではなく、一部に尖り、無駄の計測ができる
- 視認性が高い

## マネタイズポイント

- １要素に縛り、可視化・分析・無駄の提案をしてくれる（無料、サブスク・外食など）
  - 対象カテゴリ：1つだけ選択（サブスク or 外食など）
  - 登録件数：月30件まで
  - 過去データ：3ヶ月分表示
  - AI提案：月4回まで
- 複数カテゴリになる場合（有料）
  - 複数カテゴリ（サブスク＋外食 など）
  - 登録件数：無制限
  - 過去データ：無制限
  - AI提案：毎週レポート＋「今月のムダランキング」など
